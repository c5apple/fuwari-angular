W3C Web Components をフレームワーク化したもの。

SPA（Single-Page Application）を作ることができる。



W3C Web Components って何？

Web を部品化するしくみ。

  • Custom Element … 独自タグを定義する
  • HTML Template … HTMLを部品化する
  • HTML Imports … 部品化したHTMLを取り込む
  • Shadow DOM（Scoped CSS） … DOMをカプセル化する

W3C Web Components のイメージ


SPA（Single-Page Application）って何？

ひとつのページで完結したWebアプリケーション。

ページ遷移はせず、DOM操作により切り替える。

サーバとのやりとりは、REST や WebSocket などで行う。


対比されるのは、MPA (Multi-Page Application）。

MPA は従来のWebアプリケーション。
